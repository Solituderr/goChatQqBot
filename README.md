## chatgpt机器人（qq版）

### 工具

- python3.8
- golang 1.19 + echo + requests
- mysql 8.20
- **docker 20.2**
- 阿里云香港服务器 2c2g

- **chatgpt** **api**

- **cqhttp**（基于miraiGo）



### 功能

- [x] chatgpt 接入qq对话
- [x] 官方的记忆功能（官方api限制总字数）
- [x] 清空数据功能
- [x] 授予某用户权限
- [x] 私聊功能（需要管理员添加权限）
- [x] 架构完善和改进
- [x] 预设人设功能，可以添加人设（不完善）
- [ ] 多线程，集成
